# MOHA STORE DZ

موقع إلكتروني احترافي للتجارة الإلكترونية للملابس باستخدام HTML + CSS + JavaScript.

## التشغيل

1. افتح المجلد المشروع.
2. شغّل خادمًا محليًا من خلال:
   - `python -m http.server 8000`
3. افتح المتصفح على: `http://127.0.0.1:8000/index.html`

## الميزات

- صفحة رئيسية احترافية
- صفحة منتجات مع بحث وتصفية
- صفحة تفاصيل المنتج
- صفحة سلة مع تحديث الكمية والحذف
- صفحة تسجيل دخول وإنشاء حساب
- صفحة دفع
- وضع ليلي
- إشعارات عند إضافة للسلة
- حفظ السلة في LocalStorage
